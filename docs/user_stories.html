1. Login
    • Description
      As a User,
      I want to login in
      So that system aware who am I for my available operations.

    • Acceptance Criteria
      • Test 1:
        Given: User 'admin', password '123123' is an registered user.
        When: User login with credential  'admin' / '123123'.
        Then: Home page is showed with simple text "Home page".

      • Test 2:
        Given: User 'someone', password 'abcabc' is not an registered user.
        When: User login with credential  'someone' / 'abcabc'.
        Then: Login page is showed with error  message "Login failure".

      • Test 3:
        Given: None
        When: Password or user name is not entered.
        Then: Sign in button is disabled.
